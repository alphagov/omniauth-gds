# Omniauth::Gds

Omniauth strategy for GDS oauth2 provider

## Installation

Add this line to your application's Gemfile:

    gem 'omniauth-gds'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install omniauth-gds
